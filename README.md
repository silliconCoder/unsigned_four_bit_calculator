# 4-Bit (Unsigned) Calculator in Verilog

## Overview

This project implements a 4-bit (unsigned) calculator using Structural Verilog.

## Features

* 4-bit Addition
* 4-bit Subtraction
* Ripple Carry Adder based design
* Structural Modeling
* Synthesizable RTL

## Design Modules

* Full Adder
* Ripple Carry Adder
* Binary Subtractor
* Top Module: 4_BIT_CALCULATOR

## Tools Used

* Xilinx Vivado 2025.2
* Verilog HDL
* GitHub
